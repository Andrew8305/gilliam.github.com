Gettings Started using Vagrant

Note: Gilliam is very much work in progress. Please don't expect too much

The easiest way to try out Gilliam is to run it in Vagrant. Follow the steps outlines below and you will end up with a running Gilliam installation.

STEP 1: Install Vagrant

Go to http://www.vagrantup.com and install the latest release.

STEP 2: Clone the Gilliam Vagrant Repository

$ git clone https://github.com/gilliam/vagrant.git

STEP 3: Launch Vagrant

This will bootstrap your Gilliam installation. When done (it may take a while), everything is up and running.

vagrant up

STEP 4: Create your first formation

SSH into the virtual machine and issue the following commands:

$ git clone https://github.com/gilliam/example-app.git
$ cd example-app
$ gilliam-cli create example

STEP 5: Deploy

In this step the example app will be built, released and scaled.

$ gilliam-cli deploy
...
released 1
$ gilliam-cli scale 1 www=1

STEP 5: Route Traffic to Service

Here we tell the router to send traffic to the newly scaled service.

$ gilliam-cli route :/{rest:.*?} www.example.service/{rest}

STEP 6: Profit

Direct your browser to http://192.168.33.10:8080/ and you should see some text that indicated that things are working.
